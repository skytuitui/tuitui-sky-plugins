# 光遇攻略插件集合

这是一个为云崽机器人（Yunzai-Bot）开发的光遇游戏辅助插件仓库，提供光遇国际服的任务引导、数据查询等功能，持续更新中。

## 插件列表

### 1. 光遇国际服攻略插件（sky-guide）
- **功能**：查询光遇国际服每日任务、季节活动、先祖位置等攻略数据
- **适用版本**：云崽v3及以上版本
- **更新状态**：正常维护中

## 使用说明

### 命令列表
| 命令格式               | 功能描述                     | 示例                     |
|------------------------|------------------------------|--------------------------|
| `光遇攻略 每日任务`    | 查询当天国际服每日任务指南   | `光遇攻略 每日任务`      |
| `光遇攻略 季节活动`    | 查看当前季节的活动详情       | `光遇攻略 季节活动`      |
| `光遇攻略 先祖位置`    | 获取最新先祖位置及兑换表     | `光遇攻略 先祖位置 雨林` |
| `光遇攻略 帮助`        | 显示所有可用命令             | `光遇攻略 帮助`          |

### 注意事项
- 数据每日同步一次，确保攻略信息时效性
- 若查询无结果，可能是数据正在更新中，请稍后再试
