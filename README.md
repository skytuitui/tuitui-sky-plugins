# 我的私人 Yunzai 插件集合（光·遇专区）

私人插件库，仅限个人使用，所有插件基于 TRSS-Yunzai / Yunzai-Bot 开发。

## 当前包含插件

| 插件文件夹          | 功能名称          | 主要命令                  | 说明                          |
|---------------------|-------------------|---------------------------|-------------------------------|
| sky-height-cn       | 光遇身高查询      | `#身高` 或自定义          | 查询国服角色身高调整          |
| sky-height-intl     | 光遇国际服身高    | `#国际身高` 或自定义      | 查询国际服角色身高调整        |
| sky-recall          | 消息撤回          | 系统命令或自定义触发      | 快速撤回机器人发送的消息      |
| sky-guofu-fuke      | 国服复刻日程      | `国服复刻`                | 返回最新国服复刻图片          |
| sky-guofu-guide     | 国服攻略          | `国服攻略` 或其他关键词   | 返回国服活动/任务攻略图       |
| sky-intl-guide      | 国际服攻略        | `国际服攻略` 或其他关键词 | 返回国际服活动/任务攻略图     |
